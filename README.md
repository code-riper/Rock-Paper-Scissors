# Rock Paper Scissors

A simple Rock Paper Scissors game built with JavaScript.

## Features
- Random computer choice
- Human input using prompt
- 5 round game
- Score tracking
- Console-based gameplay

## Technologies Used
- HTML
- JavaScript

## How to Run
1. Open `index.html` in browser
2. Open browser console
3. Play the game

## Project Status
Currently console-based version.
GUI version will be added later.
